Project

Project Title: Minimization of transmission losses with UPFC using PSO technique
Tools: MAT Lab

Description: One of the major challenges is to reduce the transmission losses. Minimization of transmission losses is important because it leads to more economic operation of a power system. The main aim of this Project is to reduce the loss occurrence during transmission over long distance from different areas.

Steps involved in this project are: • Run Load Flow analysis problem (NRLF) in MAT lab -To find total losses
• Modeling of Unified Power Flow Controller (the optimal setting of UPFC) using Particle Swarm Optimization (PSO)
• Incorporate UPFC with the help of a Particle Swarm Optimization technique in a Standard IEEE 30 bus system thereby to reduce transmission losses.
